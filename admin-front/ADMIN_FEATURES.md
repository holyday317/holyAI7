# Holy7 管理后台功能说明

## 项目概述

Holy7 管理后台是一个功能完善的数据库管理控制台，提供用户管理、数据分析、系统监控等功能，帮助管理员高效管理 AI 助手系统的各项数据。

## 主要功能模块

### 1. 仪表板 (Dashboard)

**页面路径**: `/`

**功能特性**:
- 📊 实时数据统计卡片（用户数、会话数、消息数、书签数）
- 🎨 渐变色欢迎横幅，展示系统概览
- ⚡ 快捷操作入口，快速跳转到常用功能
- 📈 系统状态监控（版本、运行时间、数据库状态、API状态）
- 📋 数据表列表，支持查看和管理各表数据

**技术亮点**:
- 响应式布局，适配不同屏幕尺寸
- 实时数据刷新机制
- 精美的渐变色设计

### 2. 用户管理

**页面路径**: `/users`

**功能特性**:
- 👥 用户列表展示，支持分页浏览
- 🔍 用户搜索（按用户名、手机号）
- 📊 用户活跃度标签（活跃、中等活跃、不活跃）
- 📈 用户统计信息（会话数、书签数）
- 👁️ 用户详情查看（包含用户的会话记录）
- 🗑️ 用户删除功能（级联删除相关数据）

**数据展示**:
- 用户ID、用户名、手机号
- 注册时间、活跃度状态
- 会话数量、书签数量
- 最近会话记录

### 3. 会话管理

**页面路径**: `/conversations`

**功能特性**:
- 💬 会话列表展示，支持分页
- 🔍 会话搜索（按标题）
- 🎯 按用户筛选会话
- 👁️ 会话详情查看（包含完整聊天记录）
- 🗑️ 会话删除功能

**数据展示**:
- 会话ID、用户ID
- 会话标题、消息数量
- 创建时间、更新时间
- 聊天记录详情

### 4. 书签管理

**页面路径**: `/bookmarks`

**功能特性**:
- ⭐ 书签列表展示，支持分页
- 🔍 书签搜索（按内容）
- 🎯 按用户筛选书签
- 🏷️ 标签展示
- 👁️ 书签详情查看
- 🗑️ 书签删除功能

**数据展示**:
- 书签ID、用户ID、聊天ID
- 书签内容
- 标签列表
- 创建时间

### 5. 数据统计

**页面路径**: `/statistics`

**功能特性**:
- 📊 综合统计卡片（用户数、会话数、消息数、书签数）
- 📈 用户增长趋势图（支持7天/30天/90天）
- 📊 会话活跃度分析（柱状图）
- 🥧 AI模型使用分布（饼图）
- ⏰ 用户活跃时段分布（折线图）
- 📋 Prompt类型使用统计（表格+进度条）

**图表技术**:
- 使用 ECharts 5.x
- 支持交互式数据展示
- 响应式图表自适应
- 渐变色美化效果

### 6. 用户活跃度分析

**页面路径**: `/activity`

**功能特性**:
- 🎯 活跃度概览（日活、周活、月活、新增用户）
- 📈 活跃用户趋势图
- 🥧 用户活跃度分布
- ⏰ 活跃时段分析
- 📊 用户留存率分析
- 🏆 活跃用户TOP20排行榜

**评分系统**:
- 基于会话数、消息数、书签数
- 5星评分制度
- 实时更新活跃度评分

### 7. 系统日志

**页面路径**: `/logs`

**功能特性**:
- 📋 日志列表展示，支持分页
- 🔍 日志搜索（按内容）
- 🎯 按日志级别筛选（INFO/WARN/ERROR）
- 🎨 不同级别日志颜色区分
- 👁️ 日志详情查看
- 📥 日志导出功能（JSON格式）

**日志信息**:
- 时间戳、日志级别
- 请求方法、请求路径
- IP地址、用户代理
- 消息内容、完整日志

### 8. 系统配置

**页面路径**: `/config`

**功能特性**:

#### AI模型配置
- 默认模型选择（DeepSeek、OpenAI GPT-4/3.5、Claude）
- API密钥管理
- API基础URL配置
- 最大Token数设置
- 温度参数调整
- 流式响应开关

#### Prompt配置
- 默认Prompt类型选择
- 自定义Prompt模板
- Prompt增强功能开关

#### 系统设置
- 系统名称配置
- 注册功能开关
- 最大会话数限制
- 最大书签数限制
- 会话保留天数
- 日志记录开关
- 日志级别设置

#### 安全配置
- JWT认证开关
- JWT过期时间设置
- 速率限制开关
- 速率限制阈值
- IP白名单配置

#### 数据库配置
- 自动备份开关
- 备份间隔设置
- 备份数量限制
- 备份路径配置
- 手动备份功能
- 旧数据清理功能

#### 系统信息
- 系统版本
- Node版本
- 数据库类型和大小
- 运行时间
- 内存使用情况

**快速操作**:
- 刷新系统信息
- 重启服务
- 清除缓存
- 导出配置

### 9. 数据表管理

**页面路径**: `/table/:tableName`

**功能特性**:
- 📋 表结构查看
- 🔍 数据搜索
- ➕ 新增记录
- ✏️ 编辑记录
- 🗑️ 删除记录
- 📄 分页浏览

**字段类型支持**:
- 文本字段（text、char、varchar）
- 数字字段（int、real、float、double）
- 密码字段（自动隐藏）

## 技术栈

### 前端框架
- **Vue 3.4**: 渐进式JavaScript框架
- **Vue Router 4.2**: 官方路由管理器
- **Element Plus 2.5**: Vue 3组件库
- **ECharts 5.4**: 数据可视化库
- **Axios 1.6**: HTTP客户端

### 开发工具
- **Vite 5.0**: 新一代前端构建工具
- **@vitejs/plugin-vue 5.0**: Vite的Vue插件

### 后端API
- **Express.js**: Node.js Web框架
- **SQLite**: 轻量级数据库

## API接口说明

### 基础路径
```
http://106.54.34.24:3000/api/admin
```

### 主要接口

#### 用户管理
- `GET /users` - 获取用户列表
- `GET /users/:id` - 获取用户详情
- `DELETE /users/:id` - 删除用户
- `GET /users/:id/conversations` - 获取用户的会话列表

#### 会话管理
- `GET /conversations` - 获取会话列表
- `GET /conversations/:id` - 获取会话详情
- `GET /conversations/:id/chats` - 获取会话的聊天记录
- `DELETE /conversations/:id` - 删除会话

#### 书签管理
- `GET /bookmarks` - 获取书签列表
- `DELETE /bookmarks/:id` - 删除书签

#### 统计分析
- `GET /statistics` - 获取统计数据
- `GET /activity` - 获取活跃度统计

#### 系统日志
- `GET /logs` - 获取系统日志

#### 系统配置
- `GET /config` - 获取系统配置
- `POST /config` - 保存系统配置
- `GET /system/info` - 获取系统信息
- `POST /system/backup` - 备份数据库

#### 数据表管理
- `GET /tables` - 获取所有表名
- `GET /stats` - 获取数据库统计信息
- `GET /tables/:table/schema` - 获取表结构
- `GET /tables/:table/data` - 获取表数据
- `GET /tables/:table/search` - 搜索记录
- `POST /tables/:table/records` - 创建记录
- `PUT /tables/:table/records/:id` - 更新记录
- `DELETE /tables/:table/records/:id` - 删除记录

## 注意事项

1. **API地址配置**: 需要根据实际部署环境修改 `src/utils/request.js` 中的API基础路径

2. **跨域问题**: 如遇到跨域问题，需要在后端配置CORS或使用代理

3. **数据量限制**: 大数据量表格建议使用虚拟滚动或服务端分页

4. **图表性能**: ECharts图表在数据量过大时可能出现性能问题，建议合理控制数据量

5. **浏览器兼容性**: 建议使用现代浏览器（Chrome、Firefox、Edge、Safari）
